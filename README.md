# 🎬 Movie Recommendation System

---

## 📌 Project Overview

This project is a Hybrid Movie Recommendation System built using Machine Learning techniques.  
It recommends movies to users based on:

- 🎯 Collaborative Filtering (user behavior similarity)
- 🎭 Content-Based Filtering (genre similarity)

The system is deployed using Streamlit to provide an interactive web application interface.

This project demonstrates real-world recommendation engine architecture used by platforms like Netflix and Amazon.

---

## 🚀 Features

- ✅ Collaborative Filtering using Cosine Similarity  
- ✅ Content-Based Filtering using TF-IDF Vectorization  
- ✅ Hybrid Recommendation Logic  
- ✅ Interactive Streamlit Web Interface  
- ✅ Top 5 Similar Movie Suggestions  
- ✅ Clean and Modular Code Structure  

---

## ⚙️ How It Works

### 1️⃣ Collaborative Filtering

- Created User-Item Rating Matrix using Pivot Table  
- Applied Cosine Similarity  
- Recommended movies based on similar user rating patterns  

### 2️⃣ Content-Based Filtering

- Extracted movie genres  
- Applied TF-IDF vectorization  
- Computed similarity based on genre features
